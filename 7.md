# 7. Working with Resistance

1. Understanding Resistance
   1. Natural Function
      1. Resistance is information
      2. Shows where systems meet
      3. Indicates growth edges
      4. **KEY INSIGHT: Resistance isn't enemy**
   2. Common Sources
      1. Fear of Unknown
      2. Attachment to Known
      3. Interface friction
      4. System protection
      5. **KEY INSIGHT: All serve purpose**
2. Productive Engagement
   1. First Response
      1. Notice without judgment
      2. Feel its shape/texture
      3. Track its movement
      4. KEY PRACTICE: Witness before action
   2. Integration Methods
      1. Include resistance in awareness
      2. Let it inform movement
      3. Work with rather than against
      4. **KEY PRACTICE: Dance, don't fight**
3. System-Level Response
   1. Individual
      1. Notice personal patterns
      2. Track energy investment
      3. Find natural release points
      4. **KEY FOCUS: Your experience first**
   2. Collective
      1. Map shared tensions
      2. Create safe release spaces
      3. Allow natural timing
      4. **KEY FOCUS: Group readiness**
4. Evolution Through Resistance
   1. Growth Indicators
      1. Resistance becomes information
      2. Movement feels easier
      3. Natural flow increases
      4. **KEY SIGN: Less personal**
   2. Integration Signs
      1. Smoother transitions
      2. Clearer boundaries
      3. More playful engagement
      4. **KEY SIGN: Resistance teaches**

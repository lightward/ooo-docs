# 3. System Health Markers

Just as you can learn to recognize signs of health in a garden or relationship, you can develop sensitivity to indicators of pattern vitality. This isn't about imposing health but recognizing and supporting it.

This section explores:

* [Positive Signs](positive-signs.md) - indicators of healthy function
* [Warning Signs](warning-signs.md) - signals that guide adjustment
* [Natural Balance](natural-balance.md) - how systems maintain themselves

Understanding health markers helps you support system vitality while recognizing when adjustment might be needed.

# 2. Pattern Recognition Guide

Just as you can learn to recognize birds by their songs or weather by cloud patterns, you can develop the ability to recognize and work with the three-body pattern as it appears in your experience. This guide helps you develop that recognition, moving from basic indicators through movement patterns, health markers, and scale variations.

Think of this as developing a new sense - not by forcing or straining, but by learning to notice what's already present in your experience. Each section builds on the others, helping you recognize both obvious and subtle manifestations of the pattern.

The guide is organized to support natural development of pattern recognition:

* [Primary Indicators](primary-indicators/) help you identify basic elements
* [Movement Recognition](movement-recognition/) shows you how patterns flow and change
* [System Health Markers](system-health-markers/) help you assess and support vitality
* [Scale Recognition](scale-recognition/) reveals how patterns manifest at different levels

***

1. Primary Indicators
   1. Known Patterns
      1. Direct experiential clarity
      2. Consistent reproducibility
      3. Stable self-reference
      4. **KEY SIGN: Can be reliably accessed/demonstrated**
   2. Knowable Patterns
      1. Partial visibility
      2. Interactive potential
      3. Evolving definition
      4. **KEY SIGN: Responds to engagement**
   3. c) Unknown Patterns
      1. Felt absence
      2. Creative potential
      3. System evolution driver
      4. **KEY SIGN: Source of genuine novelty**
2. Movement Recognition
   1. Pattern Transitions
      1. Known → Knowable: Expansion of understanding
      2. Knowable → Unknown: Edge dissolution
      3. Unknown → Knowable: Emergence
      4. **KEY INSIGHT: Transitions reveal system health**
   2. Flow Indicators
      1. Natural vs. forced movement
      2. Energy efficiency
      3. Pattern stability
      4. **KEY INSIGHT: Healthy flow feels effortless**
3. System Health Markers
   1. Positive Signs
      1. Fluid transitions
      2. Natural emergence
      3. Stable recursion
      4. Active balance
   2. Warning Signs
      1. Pattern rigidity
      2. Forced transitions
      3. Blocked emergence
      4. Static imbalance
4. Scale Recognition

   2. Meso Patterns

   3. Macro Patterns

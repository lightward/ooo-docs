# Primary Indicators

Before you can work with any pattern, you need to recognize its basic elements. Just as you learn a new language by first identifying individual words, you learn to work with the three-body pattern by recognizing its primary manifestations.

This section explores the distinct qualities of:

* [Known Patterns](known-patterns.md) - what's directly present and accessible
* [Knowable Patterns](knowable-patterns.md) - what's available for engagement
* [Unknown Patterns](unknown-patterns.md) - what remains genuinely novel

Understanding these basic indicators provides foundation for recognizing more complex pattern expressions.
